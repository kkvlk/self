  • Riak
  • BigTable
  • Elasticsearch
  • Google Compute Cloud

Five hundred millions of real estate records, with real time, corellative search. Weekly updates from multiple sources. Greedy API consumers. That was the challenge in a nutshell.

We inherited a PHP app, tons of fixed-width data files, and constantly crashing MySQL server. We ended up with RESTful API handled by Java on Google App Engine, Elasticsearch cluster serving corellation queries, BigTable and Data Pipelines processing the updates, and sharded, redundant MySQL databases serving the data in real time.

This system, despite dealing with boring real estate data, was by far my biggest challenge, and most valuable production experience so far. It was undeniably the biggest one as well.
